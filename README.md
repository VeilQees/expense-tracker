Простое веб-приложение для учета расходов.
Позволяет добавлять траты, сортировать их, удалять и смотреть общую сумму.


Сортировка:

по дате
по сумме
по категории

Удаление расходов
Итоги по категориям

REST API для получения данных

Технологии
Java 17+
Spring Boot
Spring Data JPA
Thymeleaf
H2 / PostgreSQL (в зависимости от конфигурации)
Maven
